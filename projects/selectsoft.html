Selectsoft

Software MVP a la medida cuyo fin es servir como herramienta didáctica para aprendices del área de Talento Humano, diseñado para integrar procesos relacionados con la selección de personal en ofertas de trabajo; desde la creacion de requisiciones hasta la postulacion de candidatos y evaluacion de hojas de vida de acuerdo a parametrizaciones de cada reclutador. Este sistema fue creado como parte de la formación en el programa de Análisis y Desarrollo de Software del SENA CIDE Soacha.

Tecnologías utilizadas

HTML5
CSS3
JavaScript
Laravel
MySQL

Habilidades aplicadas

  • Desarrollo Frontend
  • Backend con Laravel
  • Modelamiento de bases de datos
  • Diseño de diagramas UML
  • Metodologias agiles (SCRUM)
  • Analisis de requerimientos
